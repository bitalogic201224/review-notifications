Increase customer engagement by notifying users about recent reviews with our customizable review notification plugin.

== Description ==
The Review Notification plugin is a powerful tool designed to increase customer trust and drive engagement by showcasing recent reviews directly on your WordPress site. With customizable display settings, this plugin lets you highlight authentic user feedback in an elegant and attention-grabbing way.


### Features
- Display recent reviews in a stylish notification popup.
- Fully customizable design to match your site's branding.
- Easy-to-use settings for managing review frequency and visibility.
- Mobile-responsive and optimized for performance.
- Boost credibility and conversions by leveraging social proof.


= How to use Review Notifications =
- First, install the Review Notifications plugin from the WordPress dashboard.
- Add the Review Notifications from the block category called "widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

### From Gutenberg Editor:
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Review Notifications**
3. Click on the **Review Notifications** to add the block

### Download & Upload:
1. Download the **Review Notifications** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`review-notifications.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

### Manually:
1. Download and upload the **Review Notifications** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
